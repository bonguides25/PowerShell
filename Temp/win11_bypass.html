Windows 11 OOBE Methods

Quick ways to continue setup with a local account or offline path.

Before You Run

Press Shift + F10 (or Fn + Shift + F10) to open Command Prompt during Windows Setup (OOBE).

Method 1 · Local-only startup

Start local-only setup directly

start ms-cxh:localonly
Method 2 · Built‑in bypass

Fastest way to trigger the offline/local path

oobe\bypassnro
shutdown /r /t 0
Method 3 · Registry BypassNRO

Enable "No Network" route via registry

reg add HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\OOBE /v BypassNRO /t REG_DWORD /d 1 /f
shutdown /r /t 0
Method 4 · Hide Online Account + Wi‑Fi

Suppress Microsoft Account and wireless setup screens

reg add HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\OOBE /v HideOnlineAccountScreens /t REG_DWORD /d 1 /f
reg add HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\OOBE /v HideWirelessSetupInOOBE /t REG_DWORD /d 1 /f
shutdown /r /t 0
Method 5 · Local admin + relaunch OOBE

Create a local admin, elevate, and restart OOBE

net user "username" /add
net localgroup "Administrators" "username" /add
cd oobe && msoobe && shutdown -r
Pro tip: Unplug Ethernet or toggle airplane mode if Wi‑Fi prompts keep appearing.
